Utilisation :

Encodage d'un fichier texte en utilisant Huffman lite :
    python3 main.py -e path_to_file
Le résultat est donné sous la forme d'un fichier encoded.txt écrit dans le même répertoire que main.py

Décodage d'un fichier texte en utilisant Huffman lite:
    python3 main.py -e path_to_file
Le résultat est écrit directement dans la sortie standard. Pour le récupérer dans un fichier vous pouvez ajouter "> votre_fichier_de_sortie.txt" à la fin de la ligne.
