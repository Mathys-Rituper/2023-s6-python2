Utilisation :

Encodage d'un fichier texte en utilisant Huffman standard :
    python3 main.py -e path_to_file
Le résultat est donné sous la forme d'un fichier texte écrit dans le même répertoire que le fichier original

Décodage d'un fichier texte en utilisant Huffman lite:
    python3 main.py -e path_to_file
Le résultat est écrit dans un fichier texte dans le même répertoire que le fichier original

Quelques fichiers pouvant servir d'exemple à encoder et décoder peuvent être trouvés dans le répertoire 4-dataset-test
